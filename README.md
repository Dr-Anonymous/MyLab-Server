### 1. Download
Go to the [Releases Page](../../releases/latest) and download **`LIMS.life.zip`**.

### 2. Install
1. Extract the zip file to a folder (e.g., `C:\LIMS.life`).
2. **Double-click `start.vbs`**.
   - Starts the server in the background.
   - **Automatically adds itself to Windows Startup**.
3. Open `http://localhost:3000` to setup.

### 3. Stop / Uninstall
- Double-click **`stop.vbs`** to stop the server and remove auto-start shortcuts.
